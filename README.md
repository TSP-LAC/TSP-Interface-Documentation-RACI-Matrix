# TSP RACI Governance Matrix

A data-driven, responsive web application to visualize the RACI Governance Matrix for SAP SuccessFactors Integration Projects.

## Features
- **Project Governance**: Clearly defined roles and responsibilities (RACI).
- **Interactive Filters**: Filter by document section or focus role.
- **Hover Documentation**: Definitions of R, A, C, and I available on hover.
- **TSP Branding**: Corporate identity integrated into the design.
- **Easy Maintenance**: Content is managed entirely via `raci-data.js`.

## Deployment
This project is hosted on GitHub Pages. To view the live application, visit the GitHub Pages URL for this repository.

## Maintenance
To update the matrix:
1. Edit `raci-data.js`.
2. Commit and push the changes to GitHub.
3. GitHub Pages will automatically redeploy the latest version.
